Go to Google!

Google Maps for mobile has a view camera. You see the map through the camera. This plugin provides animateCamera() and moveCamera() methods.

  • animateCamera()
  • moveCamera()

Map.animateCamera() method changes the camera view with animation.

JavaScript

var GOOGLE = new plugin.google.maps.LatLng(37.422858, -122.085065); map.animateCamera({ 'target': GOOGLE, 'tilt': 60, 'zoom': 18, 'bearing': 140 });

Map.moveCamera() method changes the camera view without animation.

JavaScript

var GOOGLE = new plugin.google.maps.LatLng(37.422858, -122.085065); map.moveCamera({ 'target': GOOGLE, 'tilt': 60, 'zoom': 18, 'bearing': 140 });
